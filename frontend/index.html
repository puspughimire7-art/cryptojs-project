Quick Guide

How to demo this project clearly in class.

1. Sender writes a message and secret key. 2. Click Encrypt Message. 3. Original text and key are cleared automatically. 4. Copy the encrypted text and share it. 5. Receiver pastes the encrypted text manually. 6. Receiver enters the same key and clicks Decrypt Message. 7. Receiver encrypted text and key are cleared after successful decryption. 8. Add the encrypted message to blockchain. 9. View Blockchain and Validate Blockchain.
